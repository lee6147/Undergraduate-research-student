# R&D 센터 구축 방안

ERC(2016) / CRC(2015) 선도연구센터지원사업 제안서 기반 종합 정리

## 📋 프로젝트 개요

R&D 센터 구축을 위한 종합 전략 문서를 시각화한 인터랙티브 웹페이지입니다. 센터 구축의 배경, 비전, 구조, 로드맵, 운영 방안을 체계적으로 정리하여 제시합니다.

## 🎯 핵심 질문

**"이 컨소시엄이 아니면 이 문제를 못 푼다"**

R&D 센터 구축의 본질은 다음 4가지 질문에 대한 명확한 답변입니다:
- 빌드업 = 왜 우리인가
- 체계 = 어떻게 굴리는가
- 로드맵 = 매년 뭐가 나오는가
- 스토리 = 왜 지금인가

## 📚 주요 내용 구성

### 1. 배경 (Background)
- 왜 R&D 센터를 만드는가
- 기술 트렌드와 시장 요구 분석
- 센터 설립의 필요성 및 당위성

### 2. 비전 (Vision)
**3대 핵심 이슈:**
- 기술 이슈: 관련 산업의 기술적 난제
- 시장 이슈: 시장 변화와 기회 요인
- 사회 이슈: 사회적 필요성과 파급효과

### 3. 빌드업 (Building Blocks)
**왜 우리인가?**
- 학문적 우수성: 논문, IF, 특허 실적
- 기술역량: 보유 기술 및 시설
- 인적 네트워크: 기관 연계와 협력 체계
- 사회적 신뢰: 기존 성과와 실적

### 4. 핵심 연구팀 구성 (Structure)
- **4개 핵심 연구팀** 구성
- 각 팀별 역할과 연계 방식
- 융복합 시너지 창출 구조
- 자문위원회 및 평가위원회 운영

### 5. 3단계 5년 로드맵 (Roadmap)

#### 1단계 (1-2년차): 기반 구축
- 연구팀 구성 및 장비 도입
- 기초 실험 및 개념 검증
- 산학 네트워크 구축
- **산출물**: 논문 3편, 특허 출원 2건, 학회 발표

#### 2단계 (2-3년차): 본격 연구
- 핵심 알고리즘 개발
- 시제품 제작 및 검증
- 기업 협력과제 추진
- **산출물**: 논문 5편, 특허 출원 4건, 기술이전 1건

#### 3단계 (4-5년차): 사업화
- 실증 플랫폼 구축
- 상용화 단계 기술 개발
- 국제 협력 확대
- **산출물**: 논문 8편, 특허 등록 3건, 기술이전 2건, 사업화 1건

### 6. 센터 운영 방식 (Operations)

#### 의사결정 구조
- **일반**: 자문위원회 자문 → 운영위원회 결정
- **중요**: 예산변경, 인력교체 → 핵심연구원 회의 → 운영위 의결
- **평가**: 4개 팀 운영보고서 → 평가위원회 → 운영위 의결

#### 정기 활동
- **2개월**: 실적보고 — 자체평가 확인/점검
- **분기**: 기술교류회 — 최신 동향/선도기술 공유
- **반기**: 센터 워크숍 — 성과점검/기관연계
- **연간**: 자체평가 — 정성/정량 평가 → 포상/경고/반영

#### 5대 평가 항목
1. **연구**: 논문 게재, 특허 출원/등록, 기술이전
2. **산학**: 기업 기술지원, 협력과제 도출
3. **국제**: 국제 공동과제, 인력 교류
4. **홍보**: 홈페이지, 심포지엄, 소식지
5. **교육**: 인력양성, 전문가 양성, 교육세미나

#### 연구 전략
- **방법**: Tech Push + Demand Pull 병행
- **관점**: 기술 중심이 아닌 사용자/사회 현상 고려
- **연계**: 산-학-연 인력 수요 반영, 수요지향 인력양성
- **지원**: 참여기업 기술한계 극복 분석지원 + 기술지도

## 💡 핵심 메시지

> 센터 구축의 핵심은 **장비가 아니라 체계**이고  
> **기술이 아니라 스토리**입니다

"이 팀이 아니면 이 문제를 못 푼다"는 확신을 주는 구조.  
매년 확실한 산출물이 나오는 단계적 접근.  
기술을 뼈대로, 스토리를 살로 입히는 것.  
그것이 R&D 센터 구축의 본질입니다.

## 🎨 웹페이지 특징 & 애니메이션 효과

### 🌟 히어로 섹션 시각 효과
- **애니메이션 그리드 배경**: 펄스 효과가 있는 50x50px 그리드 패턴 (6초 주기)
- **45개 파티클 시스템**: 5가지 컬러(cyan, green, blue, purple, yellow)의 파티클이 하단에서 상단으로 떠오름 (7초 주기)
- **블러 Orb 애니메이션**: 100px 블러 처리된 대형 구체가 12초 주기로 이동하며 분위기 연출
- **마우스 패럴랙스**: 마우스 움직임에 반응하여 Orb들이 부드럽게 따라움 (레이어별 차등 속도)
- **Radial Gradient 배경**: 3개의 방사형 그라디언트(cyan, green, purple)가 겹쳐진 배경

### ✍️ 타이핑 & 텍스트 효과
- **라이브 타이핑 효과**: "이 컨소시엄이 아니면 이 문제를 못 푼다" 텍스트가 60ms 간격으로 타이핑됨
- **커서 블링크**: 1초 주기로 깜빡이는 cyan 색상 커서
- **그라디언트 텍스트**: cyan → green 그라디언트가 적용된 강조 텍스트
- **페이드 인/아웃**: 각 요소별 순차적 등장 (0.2초 간격 stagger)

### 📊 스크롤 기반 인터랙션
- **4가지 Reveal 애니메이션**:
  - `rv`: 하단에서 위로 페이드인 (60px 이동)
  - `rv-l`: 좌측에서 페이드인 (80px 이동)
  - `rv-r`: 우측에서 페이드인 (80px 이동)
  - `rv-s`: 스케일 애니메이션 (0.85 → 1.0)
- **Intersection Observer**: 뷰포트 진입 시 자동 트리거 (8% threshold, -20px rootMargin)
- **스무스 스크롤**: 네비게이션 클릭 시 부드러운 스크롤 이동

### 📈 동적 요소들
- **카운터 애니메이션**: 숫자가 0부터 목표값까지 25ms 간격으로 증가 (50단계)
- **프로그레스 바**: 
  - 상단 고정 4px 높이 바
  - 4색 그라디언트(green → cyan → blue → purple)
  - 글로우 효과(16px + 40px 박스 섀도우)
  - 실시간 스크롤 비율 반영
- **네비게이션 슬라이드**: 
  - 500px 스크롤 후 상단에서 슬라이드 인
  - 배경 블러 (24px backdrop-filter)
  - 호버 시 언더라인 애니메이션 (scaleX transition)

### 🎯 UI/UX 인터랙션
- **호버 효과**:
  - 카드: 테두리 색상 변화 + 미묘한 상승 효과
  - 버튼: 박스 섀도우 확대 + 밝기 증가
  - 링크: cyan 색상 전환 + 하단 바 등장
- **글로우 효과**: 모든 주요 요소에 적용된 박스 섀도우 (메인 컬러별)
- **바운스 애니메이션**: 스크롤 큐 화살표 (2초 주기, 12px 이동)
- **백 투 탑 버튼**: 700px 스크롤 후 우하단 등장, 클릭 시 최상단 이동

### 🎨 색상 시스템 & 테마
- **커스텀 CSS 변수**: 10개 메인 컬러 + 5개 배경/텍스트 컬러
- **다크 테마 최적화**: 
  - 배경: #050a14 (메인), #0b1322 (서브)
  - 카드: #111d30, #162038
  - 텍스트: #eef2f7 (주), #8da2be (보조)
- **시맨틱 컬러**:
  - Green (#22c55e): 성공, 완료
  - Cyan (#06b6d4): 강조, 링크
  - Blue (#3b82f6): 정보
  - Purple (#8b5cf6): 특별
  - Yellow (#eab308): 주의, 중요
  - Orange (#f97316): 경고
  - Red (#ef4444): 긴급, 평가

### 📱 반응형 & 성능
- **반응형 레이아웃**: 모바일/태블릿/데스크톱 대응
- **requestAnimationFrame**: 스크롤 이벤트 최적화 (throttle)
- **CSS Transform & Opacity**: GPU 가속 애니메이션
- **Cubic Bezier**: 자연스러운 easing (.16, 1, .3, 1)
