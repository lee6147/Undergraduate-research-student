# Obsidian HTML Viewer

Obsidian에서 HTML 가이드 파일을 바로 열어볼 수 있는 환경입니다.

---

## 핵심 플러그인

| 플러그인 | 역할 |
|---------|------|
| **obsidian-html-plugin** (HTML Reader) | `.html` 파일을 파일 탐색기에서 인식 → 클릭으로 바로 열기 |

> Obsidian은 기본적으로 `.html` 파일을 열 수 없습니다. `obsidian-html-plugin`이 이를 가능하게 합니다.

---

## 시작하기

### 1. Obsidian 커뮤니티 플러그인에서 HTML Reader 설치

1. **설정**(좌측 하단 톱니바퀴) → **커뮤니티 플러그인**
2. **제한 모드 해제** → **찾아보기**
3. `HTML Reader` 검색 후 설치 및 활성화

### 2. HTML 파일 열기

볼트 내 아무 폴더에 `.html` 파일을 넣으면 파일 탐색기에서 클릭해서 바로 열 수 있습니다.

---

## ***** 명령어 팔레트로 파일 찾기

`Ctrl + P`를 누르면 명령어 팔레트가 열립니다. 파일명을 검색하면 HTML 파일도 바로 찾아서 열 수 있습니다.

---

## HTML 가이드 만들기

JSX/TSX 파일을 Obsidian에서 실행 가능한 단일 HTML로 변환할 수 있습니다.
AI에게 [`JSX_Obsidian_변환프롬프트_v4.4.md`](./JSX_Obsidian_변환프롬프트_v4.4.md)를 붙여넣고 JSX 파일을 주면 자동으로 변환해줍니다.
